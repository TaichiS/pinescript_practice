# 專案簡介

PineScript 是 TradingView 網站看盤多功能圖表專用的程式語言，這邊是程式碼範例的存放處，方便大家複製貼上。

我將教學的課程，分享在這個 youtube 頻道：

https://www.youtube.com/channel/UCIii8L5wtZsyVgYlbkLuH4A

# 呼叫程式碼說明書的方法

* 如果你用 PC，可以使用 Ctrl-Click
* 如果你用MAC，可以使用 Cmd-Click
叫出程式碼的使用說明書。
